# 【完全マニュアル】AIアフィリエイト自動化システム

このマニュアルは、AIを活用してアフィリエイトサイトを半自動で運営するための手順書です。指示通りに進めるだけで、専門知識がなくても収益化の仕組みを構築できます。

---

## 0. 仕組みの全体像

このシステムは、以下の無料・低コストのサービスを組み合わせて、あなたの代わりにAIがブログ記事を書き続け、アフィリエイトサイトを運営します。

- **サイト公開**: GitHub Pages（完全無料）
- **記事作成**: OpenAI API（月500〜1,000円程度の低コスト）
- **自動化**: GitHub Actions（無料）
- **収益化**: A8.netなどのASP（無料）

一度設定すれば、**毎週自動で新しい記事がサイトに追加され続ける**ため、あなたが寝ている間もサイトが成長し、収益機会が拡大します。

## 1. 必要なアカウントの作成（約15分）

まず、以下の3つのサービスに登録してください。すべて無料で始められます。

| サービス名 | 用途 | 登録URL |
|---|---|---|
| **GitHub** | サイトの保管・公開、自動化の実行 | [https://github.com/signup](https://github.com/signup) |
| **A8.net** | アフィリエイト広告の取得 | [https://www.a8.net/](https://www.a8.net/) |
| **OpenAI** | AIによる記事作成 | [https://platform.openai.com/signup](https://platform.openai.com/signup) |

### A8.netの登録Tips
- 登録時にサイト情報が必要ですが、この時点ではまだサイトがないため、**X（旧Twitter）のアカウントで登録**するか、A8.netが提供する**無料ブログ「ファンブログ」を作成**して登録するのが簡単です。
- 登録後、「クレジットカード」ジャンルの広告プログラムにいくつか提携申請しておきましょう。（例：楽天カード、三井住友カード、JCBカードなど）

## 2. GitHubリポジトリのセットアップ（約10分）

ここからは、あなたのアフィリエイトサイトの「設計図」をGitHubに設置する作業です。

1.  **新しいリポジトリを作成**
    -   GitHubにログインし、右上の「+」から「New repository」を選択します。
    -   **Repository name** に `あなたのユーザー名.github.io` と入力します。（例： `taro-yamada.github.io`）
    -   **Public** を選択し、「Create repository」をクリックします。

2.  **プロジェクトファイルをアップロード**
    -   私が提供するプロジェクトファイル（zip形式）をダウンロードし、解凍します。
    -   作成したリポジトリのページで、「Add file」→「Upload files」を選択します。
    -   解凍したフォルダの中身をすべてドラッグ＆ドロップし、「Commit changes」をクリックします。

    > これで、あなたのサイトの雛形がインターネット上に準備できました。 `https://あなたのユーザー名.github.io` にアクセスすると、サンプルサイトが表示されるはずです。（反映に数分かかる場合があります）

## 3. あなた専用の設定に書き換える（約20分）

次に、サイトをあなた自身のものにするための設定変更を行います。

### 3-1. サイト基本設定の変更

1.  GitHubのリポジトリページで、`_config.yml` というファイルを探してクリックします。
2.  右上の鉛筆アイコン（Edit this file）をクリックします。
3.  以下の2箇所をあなたの情報に書き換えます。
    -   `url`: `https://あなたのユーザー名.github.io`
    -   `author.name`: あなたのニックネーム
4.  変更が終わったら、「Commit changes」をクリックします。

### 3-2. アフィリエイトリンクの設定

1.  `scripts/generate_article.py` というファイルを開き、編集画面に移動します。
2.  ファイルの上部にある `AFFILIATE_LINKS` のセクションを探します。
3.  A8.netなどで取得した**あなた自身のアフィリエイトリンク**に差し替えます。

    ```python
    # 変更前
    AFFILIATE_LINKS = {
        "三井住友カード": "https://ck.jp.ap.valuecommerce.com/servlet/referral?sid=YOUR_SID&pid=YOUR_PID",
        "楽天カード": "https://ck.jp.ap.valuecommerce.com/servlet/referral?sid=YOUR_SID&pid=YOUR_PID2",
        # ...
    }

    # 変更後（あなたのリンクに書き換える）
    AFFILIATE_LINKS = {
        "三井住友カード": "https://px.a8.net/svt/ejp?a8mat=XXXXXX...",
        "楽天カード": "https://px.a8.net/svt/ejp?a8mat=YYYYYY...",
        # ...
    }
    ```
4.  変更が終わったら、「Commit changes」をクリックします。

## 4. AIを動かすための設定（約5分）

最後に、AI（OpenAI）を動かすための「鍵」を設定します。この鍵は他人に知られないように厳重に扱ってください。

1.  **OpenAI APIキーの取得**
    -   [OpenAIのAPIキーページ](https://platform.openai.com/api-keys)にアクセスします。
    -   「Create new secret key」をクリックし、キーをコピーします。（このキーは一度しか表示されません）

2.  **GitHubにAPIキーを登録**
    -   あなたのGitHubリポジトリのページで、「Settings」→「Secrets and variables」→「Actions」に移動します。
    -   「New repository secret」をクリックします。
    -   **Name** に `OPENAI_API_KEY` と入力します。
    -   **Secret** に先ほどコピーしたOpenAIのAPIキーを貼り付け、「Add secret」をクリックします。

## 5. 自動化の開始と確認

お疲れ様でした！これで全ての準備が整いました。

### 手動で最初の記事を生成してみる

1.  リポジトリの「Actions」タブをクリックします。
2.  左側の「記事自動生成 & デプロイ」を選択します。
3.  「Run workflow」というボタンが表示されるので、クリックして実行します。

> 実行には5〜10分かかります。完了すると、`_posts` フォルダに新しい記事ファイルが自動で追加され、あなたのサイト `https://あなたのユーザー名.github.io` にも新しい記事が公開されます。

### 今後の自動運用

この設定により、**毎週月・水・金の朝9時に、AIが自動で新しい記事を生成し、あなたのサイトに公開します。**

あなたは時々サイトをチェックして、収益を確認するだけです。

## 6. カスタマイズ（応用編）

-   **生成する記事のキーワードを変えたい場合**: `scripts/generate_article.py` の中にある `KEYWORDS` リストを編集してください。
-   **サイトのデザインを変えたい場合**: `_config.yml` の `theme` を変更したり、CSSをカスタマイズすることで可能です。（上級者向け）
