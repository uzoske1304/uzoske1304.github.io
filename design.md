# アフィリエイト自動化 全体設計

## 戦略概要

### 選定ジャンル：クレジットカード比較（金融系）
- 理由：1件成約で3,000〜10,000円の高単価報酬
- 競合は多いが、特定のロングテールキーワードで上位表示可能
- 申し込みハードルが低く（無料で申し込める）、成約率が高い

### 使用するASP
- **A8.net**（業界最大手・登録無料・サイトなしでも登録可）
- **もしもアフィリエイト**（Amazon/楽天も扱える・登録無料）

### サイト公開方法
- **GitHub Pages**（完全無料・独自ドメイン対応・商用利用OK）
- ドメイン：GitHub提供の `username.github.io` を使用（費用ゼロ）

## 自動化の仕組み

```
[OpenAI API] → [記事自動生成スクリプト(Python)] → [Markdownファイル]
                                                          ↓
                                               [GitHub Actions]
                                                          ↓
                                          [GitHub Pages に自動デプロイ]
                                                          ↓
                                              [アフィリエイトサイト公開]
```

### 自動化フロー詳細

1. **コンテンツ生成（週1〜3回）**
   - Pythonスクリプトが OpenAI API を呼び出し
   - SEOキーワードに基づいた記事を自動生成
   - アフィリエイトリンクを自動挿入
   - Markdownファイルとして保存

2. **自動デプロイ（GitHub Actions）**
   - GitHubにpushすると自動でサイトが更新
   - 静的サイトジェネレータ（Jekyll）でHTML変換
   - GitHub Pagesに自動公開

3. **スケジュール実行（GitHub Actions cron）**
   - 毎週決まった曜日・時間に自動で記事生成＆公開
   - 人手不要で継続的にコンテンツが増え続ける
